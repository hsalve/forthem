# ForThem Roadmap

ForThem is a mobile-first co-parenting app for busy parents who need to coordinate a child's life with less stress, fewer tools, and fewer repeated conversations.

## Product Principles

- Busy parents first: every common action should take less than 10 seconds.
- Shared where needed, private where appropriate.
- Reduce typing, tapping, scrolling, and mental load.
- Calm, premium, consumer-app feel; not court software.
- Build one polished, functional feature at a time.

## MVP Goal

Two co-parents should be able to use ForThem instead of scattered tools like Google Calendar, Splitwise, texts, and shared folders.

## MVP Features

### 1. Authentication and Setup
- Email/password login through Supabase.
- Create parent profile.
- Create family.
- Add child.
- Invite co-parent.
- Stay logged in across app restarts.

### 2. Parent and Child Profiles
- Parent can edit their own profile after setup.
- Parent can add more children after setup.
- Parent can edit child profiles after setup.

### 3. Shared Calendar
- Custody schedule.
- Add/edit/delete child-related events.
- Import existing calendar events later.
- Events can be shared or private.

### 4. Private Parent Tools
- Parent-only reminders.
- Parent-only notes.
- Personal calendar items for days the parent has the child.

### 5. Swaps
- Request swap.
- Other parent can approve or decline.
- Approved swap updates calendar.
- Optional comment/reason field.

### 6. Expenses
- Add expense.
- Split by 50/50, percentage, exact amount, or parenting-plan rule.
- Recurring weekly/monthly expenses, especially daycare.
- Track pending, approved, and settled expenses.

### 7. Documents
- Upload files.
- Organize by category: Medical, Insurance, School, Legal, Passport, Other.
- Parenting plan upload.
- Manual extracted rules first; AI extraction later.

## Post-MVP Features

- Push notifications.
- Google Calendar live sync.
- Receipt scanning.
- AI parenting plan extraction.
- AI co-parenting assistant.
- Travel request workflow.
- Packing lists for exchanges.
- School/daycare hub.
- Child timeline.

## Current Development Focus

Work one feature at a time. Keep changes small, testable, and aligned with the existing design system.
